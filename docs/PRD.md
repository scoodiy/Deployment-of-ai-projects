# 量化交易机器人 - 产品需求文档

## 1. 产品概述
量化交易机器人是一个全自动化交易平台，支持股票和加密货币的量化交易策略执行、风险管理和智能问答。

## 2. 核心功能

### 2.1 策略引擎
- 支持网格交易、马丁格尔、趋势跟踪等策略
- 用户可自定义策略并上传
- 策略调度与自动执行

### 2.2 风控引擎
- 仓位限制检查
- 每日亏损监控
- 连续亏损保护
- 熔断机制
- API健康监控

### 2.3 数据采集
- A股数据（东方财富、同花顺）
- 加密货币数据（Binance、OKX、Bybit）
- 数据清洗与标准化

### 2.4 智能问答 (RAG)
- 基于知识库的问答系统
- 意图识别与路由
- 文档检索与重排序

### 2.5 前端界面
- 实时仪表盘
- 交易下单界面
- 策略管理面板
- 风控监控面板
- 智能问答聊天

## 3. 技术架构
- 前端：React + TypeScript + Vite + Tailwind CSS
- 后端：FastAPI + SQLAlchemy + Redis + Kafka
- 策略引擎：Python + NumPy + Pandas
- 风控引擎：Python (独立服务)
- 数据采集：Python + httpx (多源适配器)
- QA Bot：Python RAG pipeline
